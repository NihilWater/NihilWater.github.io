导出专栏

功能介绍

本功能让用户将专栏（public/markdown/下的一个文件夹）发布为word。

使用流程

请首先安装 pandoc

下载链接：https://www.pandoc.org/installing.html 下载完成后需要将pandoc 配置到环境变量中

命令

node run toword [:name] [:type]
# [:name] 代表是哪个专栏
# [:type] 代表是哪个类型，比如a和p
   -- a all 表示打包所有的文章
   -- p public 表示打包所有公开的文章
# 例如 node run toword niswon a
